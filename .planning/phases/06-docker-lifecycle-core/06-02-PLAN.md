**File 1: `internal/docker/launch.go`**

Define `LaunchOpts` struct:
```go
type LaunchOpts struct {
    Headless    bool
    Prompt      string
    Rebuild     bool
    NoCache     bool
    HarnessArgs []string
}
```

Implement `Manager.Launch(ctx context.Context, opts LaunchOpts) error`:

1. Acquire lock via `cache.NewLock(m.cache.Dir())` -> `lock.Acquire()`. Do NOT defer Release — release is explicit before attach.

2. Read `m.cache.ContainerID()`. If non-empty:
   - Call `m.inspectContainerState(ctx, containerID)`. If returns `(nil, nil)`, container is stale: call `m.cleanStaleCache(ctx)` and fall through to build.
   - If inspect returns valid info, switch on `info.State.Status`:
     - `"running"`: call `m.handleRunning(ctx, info, lock, opts)` and return its result
     - `"paused"`: call `m.client.ContainerUnpause(ctx, containerID)`, release lock, attach
     - `"exited"`, `"dead"`: warn on OOM if `info.State.OOMKilled` (`fmt.Fprintf(os.Stderr, "Warning: Container was killed due to memory limit. Increase resources.memory in zone.toml.\n")`), then `m.client.ContainerRemove(ctx, containerID, container.RemoveOptions{Force: true})`, `m.removeNetwork(ctx)`, `m.cache.SetContainerID("")`, `m.cache.SetNetworkID("")`
     - `"created"`, `"restarting"`: `time.Sleep(2 * time.Second)`, then stop with `timeout := 5; m.client.ContainerStop(ctx, containerID, container.StopOptions{Timeout: &timeout})`, remove, clear cache

3. Build if needed: call `m.buildIfNeeded(ctx, opts.Rebuild, opts.NoCache)`. On error, release lock and return.

4. Create and start container: call `m.createAndStart(ctx)`. On error, release lock and return.

5. Release lock.

6. If `opts.Headless`: `fmt.Println(containerID)`, return nil.

7. Otherwise: `return m.attachInteractive(containerID, m.harnessCmd(opts), false)`

Helper methods on Manager:

`inspectContainerState(ctx, containerID)` — wraps `m.client.ContainerInspect`, returns `(*container.InspectResponse, error)`. If `errdefs.IsNotFound(err)`, returns `(nil, nil)`.

`handleRunning(ctx, info, lock, opts)` — checks config hash via `m.checkConfigHash()`. If hash differs, print warning: `"Config has changed since this container was started. Run 'zone restart --rebuild' to apply changes."` to stderr. Release lock. If headless, print container ID and return. Otherwise attach.

`checkConfigHash()` — calls `cache.ComputeHash(m.config, m.version)`, compares with `m.cache.ConfigHash()`. Returns `(changed bool, err error)`.

`cleanStaleCache(ctx)` — clear container_id, attempt network remove (swallow errors), clear network_id.

`buildIfNeeded(ctx, forceRebuild, noCache)` — if `forceRebuild`, always build. Otherwise compute hash, compare with cached hash. If same AND image still exists (verify via `m.client.ImageInspectWithRaw`), skip build. Otherwise call `m.buildImage(ctx, noCache)`.

`createAndStart(ctx)` — calls `m.createContainer(ctx, imageID)` using cached image ID, then `m.client.ContainerStart(ctx, containerID, container.StartOptions{})`, saves container ID to cache.

`harnessCmd(opts)` — builds the command slice for the harness entrypoint. Gets harness via `harness.Get(m.config.Zone.Harness, &m.config.Harness)`. Uses `h.EntrypointCommand()` as base. If `opts.Prompt != ""`, appends `h.PromptFlag()` and `opts.Prompt`. Appends `opts.HarnessArgs`.

**File 2: `internal/docker/quickstart.go`**

```go
package docker

import (
    "fmt"
    "os"
    "path/filepath"

    "github.com/peasant-labs/zone/internal/cache"
)

const minimalZoneTomlTemplate = `version = 1
harness = "%s"

# Uncomment to customize:
# [zone]
# base_image = "ubuntu:24.04"
# shell = "bash"
#
# [resources]
# memory = "4g"
# cpus = "2"
# pids_limit = 512
#
# [workspace]
# persist_home = true
`

func generateMinimalZoneToml(harness string) string {
    return fmt.Sprintf(minimalZoneTomlTemplate, harness)
}

func (m *Manager) handleZeroConfig(harnessName string) error {
    tomlPath := filepath.Join(m.repoDir, "zone.toml")
    content := generateMinimalZoneToml(harnessName)
    if err := os.WriteFile(tomlPath, []byte(content), 0644); err != nil {
        return fmt.Errorf("write zone.toml: %w", err)
    }
    if err := cache.EnsureGitignore(m.repoDir); err != nil {
        return fmt.Errorf("update gitignore: %w", err)
    }
    fmt.Fprintf(os.Stderr, "Created zone.toml with harness %q\n", harnessName)
    return nil
}
```
