**Add to `internal/docker/manager.go`:**

`Manager.Stop(ctx context.Context) error`:
1. Read `m.cache.ContainerID()`. If empty, return nil (no-op, already stopped).
2. Stop container: `timeout := 10; m.client.ContainerStop(ctx, containerID, container.StopOptions{Timeout: &timeout})`. If error and NOT `errdefs.IsNotFound(err)`, return error. If NotFound, continue (container already gone).
3. Remove container: `m.client.ContainerRemove(ctx, containerID, container.RemoveOptions{Force: true})`. Swallow NotFound.
4. Remove network: `m.removeNetwork(ctx)`.
5. Clear cache: `m.cache.SetContainerID("")`, `m.cache.SetNetworkID("")`.
6. Return nil.

`Manager.Destroy(ctx context.Context) error`:
1. Call `m.Stop(ctx)`. If error, return it.
2. Remove image: `imageID, _ := m.cache.ImageID()`. If non-empty, call `m.client.ImageRemove(ctx, imageID, image.RemoveOptions{Force: false, PruneChildren: true})`. Swallow `errdefs.IsNotFound`.
3. Remove home volume: call `m.client.VolumeRemove(ctx, homeVolumeName(m.repoDir), false)`. Swallow NotFound errors.
4. Remove all cache: `m.cache.Clean()`.
5. Return nil.

`Manager.RemoveImage(ctx context.Context) error`:
1. Read `m.cache.ImageID()`. If empty, return nil.
2. Call `m.client.ImageRemove(ctx, imageID, image.RemoveOptions{Force: false, PruneChildren: true})`. Swallow NotFound.
3. Clear cache: `m.cache.SetImageID("")`.
4. Return nil.

**Note on error handling:** Every Docker SDK call that can fail with "not found" (container already removed, image already pruned, volume doesn't exist) must be wrapped with `errdefs.IsNotFound(err)` check. Not-found errors are swallowed because the goal of Stop/Destroy is to reach a clean state — if the resource is already gone, that's the desired end state.
