  <action>
    Create `internal/docker/env.go` in package `docker` with these functions:

    **CollectForwardedEnv(patterns []string) (envVars []string, warnings []string)**
    - Iterate `os.Environ()` which returns `[]string{"KEY=VALUE", ...}`
    - For each env entry, split on first `=` to get key
    - For each pattern in `patterns`, call `filepath.Match(pattern, key)`
    - If match, append the full `"KEY=VALUE"` string to result (use a `map[string]bool` to deduplicate keys)
    - After iterating all env vars, check which patterns matched zero host vars: for each such pattern, append `fmt.Sprintf("Warning: forward_env pattern %q did not match any host environment variables", pattern)` to warnings
    - Return the collected envVars and warnings slices
    - Import: `"os"`, `"path/filepath"`, `"strings"`, `"fmt"`

    **ParseEnvFile(path string) (map[string]string, error)**
    - Read file with `os.ReadFile(path)` — return `fmt.Errorf("read env file %s: %w", path, err)` on failure
    - Create `bufio.Scanner` from `bytes.NewReader(data)`
    - For each line: `strings.TrimSpace(scanner.Text())`
    - Skip if line is empty or starts with `#`
    - Find first `=` with `strings.Index(line, "=")`; skip if `idx < 0`
    - key = `line[:idx]`, val = `line[idx+1:]`
    - Store in result map
    - Return result map, scanner.Err()
    - Import: `"bufio"`, `"bytes"`, `"os"`, `"strings"`, `"fmt"`

    **ValidateRequiredEnv(required []string, harnessName string, envFilePath string, repoDir string) error**
    - If `len(required) == 0`, return nil immediately
    - Build combined env map: start with host env via `os.Environ()` (split on first `=`, store key->true)
    - If `envFilePath != ""`: resolve path relative to repoDir via `filepath.Join(repoDir, envFilePath)` (if not already absolute); call `ParseEnvFile(resolvedPath)` — if error, return `fmt.Errorf("parse env file: %w", err)`; add all keys from .env to the combined map
    - For each required var: check if key exists in combined map. If missing, return `fmt.Errorf("required environment variable %s is not set. The %s harness needs this variable. Set it and re-run zone launch", missing, harnessName)`
    - Return nil if all present

Use `t.TempDir()` for .env file tests. Follow existing testify patterns from manager_test.go (assert.Equal, require.NoError, assert.Contains).
