    Create `internal/docker/ports.go` in package `docker` with:

    **Import:** `"fmt"`, `"strconv"`, `"strings"`, `"github.com/docker/go-connections/nat"`

    **parsePortBindings(ports []string) (nat.PortMap, nat.PortSet, error)**
    - Initialize `bindings := nat.PortMap{}`, `exposed := nat.PortSet{}`, `seenHostPorts := map[string]bool{}`
    - For each port string `p` in `ports`:
      - `parts := strings.SplitN(p, ":", 2)`
      - If `len(parts) != 2`: return error `fmt.Errorf("invalid port entry %q: expected hostPort:containerPort", p)`
      - `hostPort, containerPort := parts[0], parts[1]`
      - Call `validatePort(hostPort)` — if error: `return nil, nil, fmt.Errorf("invalid host port in %q: %w", p, err)`
      - Call `validatePort(containerPort)` — if error: `return nil, nil, fmt.Errorf("invalid container port in %q: %w", p, err)`
      - If `seenHostPorts[hostPort]`: `return nil, nil, fmt.Errorf("conflicting port binding: host port %s appears more than once", hostPort)`
      - `seenHostPorts[hostPort] = true`
      - `natPort, err := nat.NewPort("tcp", containerPort)` — if error: `return nil, nil, fmt.Errorf("parse container port %q: %w", containerPort, err)`
      - `bindings[natPort] = []nat.PortBinding{{HostPort: hostPort}}`
      - `exposed[natPort] = struct{}{}`
    - Return `bindings, exposed, nil`

    **validatePort(s string) error**
    - `n, err := strconv.Atoi(s)` — if error: `return fmt.Errorf("port %q is not a valid number", s)`
    - If `n < 1 || n > 65535`: `return fmt.Errorf("port %d is out of range (1-65535)", n)`
    - Return nil

    Create `internal/docker/ports_test.go` with tests matching the behavior list. Use `assert.Equal`, `require.NoError`, `assert.ErrorContains` (testify).

    After creating the files, run `go mod tidy` in case `github.com/docker/go-connections/nat` needs to be promoted from indirect to direct dependency.

    **proxy.go** in package `docker`:

    Import: `"fmt"`, `"os"`

    **resolveProxy(cfg *config.NetworkConfig) (httpProxy, httpsProxy, noProxy string)**
    - `httpProxy = cfg.HTTPProxy`; if `httpProxy == ""` then `httpProxy = firstEnv("HTTP_PROXY", "http_proxy")`
    - `httpsProxy = cfg.HTTPSProxy`; if `httpsProxy == ""` then `httpsProxy = firstEnv("HTTPS_PROXY", "https_proxy")`
    - `noProxy = cfg.NoProxy`; if `noProxy == ""` then `noProxy = firstEnv("NO_PROXY", "no_proxy")`
    - Return all three

    **firstEnv(keys ...string) string** — iterate keys, return first non-empty `os.Getenv(key)`; return "" if all empty

    **proxyBuildArgs(httpProxy, httpsProxy, noProxy string) map[string]*string**
    - Initialize `args := map[string]*string{}`
    - If `httpProxy != ""`: `v := httpProxy; args["HTTP_PROXY"] = &v; args["http_proxy"] = &v`
      CRITICAL: create separate local variable for each value to avoid pointer aliasing: `v1 := httpProxy; args["HTTP_PROXY"] = &v1; v2 := httpProxy; args["http_proxy"] = &v2`
    - Same pattern for httpsProxy (HTTPS_PROXY, https_proxy) and noProxy (NO_PROXY, no_proxy)
    - Return args

    **proxyEnvVars(httpProxy, httpsProxy, noProxy string) []string**
    - Initialize `var envs []string`
    - If `httpProxy != ""`: append `"HTTP_PROXY=" + httpProxy` and `"http_proxy=" + httpProxy`
    - Same for httpsProxy and noProxy
    - Return envs

    Import `"github.com/peasant-labs/zone/internal/config"` for the NetworkConfig type parameter.

    **hooks.go** in package `docker`:

    Import: `"fmt"`, `"io"`, `"os"`, `"os/exec"`

    **runHooks(cmds []string, repoDir string, failFast bool, stderr io.Writer) error**
    - For each `cmd` in `cmds`:
      - `c := exec.Command("sh", "-c", cmd)`
      - `c.Dir = repoDir`
      - `c.Stdout = os.Stdout`
      - `c.Stderr = stderr` (use passed writer, defaults to os.Stderr in production)
      - If `err := c.Run(); err != nil`:
        - If `failFast`: return `fmt.Errorf("hook %q failed: %w", cmd, err)`
        - Else: `fmt.Fprintf(stderr, "Warning: hook %q failed: %v\n", cmd, err)`
    - Return nil
