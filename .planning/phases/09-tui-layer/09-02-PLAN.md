**Part A: Add BuildWithProgress to internal/docker/build.go**

Add a new exported method on Manager that wraps the existing synchronous `buildImage` and feeds build output lines to a channel. This avoids refactoring the existing `buildImage`/`streamBuildOutput` -- it layers over them.

```go
// BuildLine carries a single line of Docker build output.
type BuildLine struct {
    Text string
}

// BuildResult carries the final build outcome.
type BuildResult struct {
    ImageID string
    Err     error
}

// BuildWithProgress starts the Docker image build in a goroutine.
// Build output lines are sent to the lines channel as they arrive.
// The final result (image ID or error) is sent to the result channel.
// Both channels are created by the caller and closed by this method.
//
// Usage: create channels, call BuildWithProgress, read lines until result arrives.
func (m *Manager) BuildWithProgress(ctx context.Context, noCache bool, lines chan<- BuildLine, result chan<- BuildResult) {
    go func() {
        defer close(lines)
        defer close(result)
        imageID, err := m.buildImageWithChannel(ctx, noCache, lines)
        result <- BuildResult{ImageID: imageID, Err: err}
    }()
}
```

Add a private `buildImageWithChannel` method that is a copy of `buildImage` but replaces the `streamBuildOutput` call with `streamBuildOutputWithChannel` that writes each line to the channel AND to the log file:

```go
func streamBuildOutputWithChannel(body io.ReadCloser, logW io.Writer, lines chan<- BuildLine) (string, error) {
    defer body.Close()
    scanner := bufio.NewScanner(body)
    var imageID string
    for scanner.Scan() {
        var msg buildMessage
        if jsonErr := json.Unmarshal(scanner.Bytes(), &msg); jsonErr != nil {
            continue
        }
        if msg.Error != "" {
            return "", fmt.Errorf("docker build: %s", msg.Error)
        }
        if msg.Stream != "" {
            fmt.Fprint(logW, msg.Stream) // still write to log
            lines <- BuildLine{Text: msg.Stream}
        }
        if msg.Aux != nil && msg.Aux.ID != "" {
            imageID = msg.Aux.ID
        }
    }
    return imageID, scanner.Err()
}
```

**Part B: Add NeedsBuild to internal/docker/launch.go**

```go
// NeedsBuild returns true if a Docker image build is required.
// It checks force rebuild, hash mismatch, and image existence.
func (m *Manager) NeedsBuild(ctx context.Context, forceRebuild bool) bool {
    if forceRebuild {
        return true
    }
    current, err := cache.ComputeHash(m.config, m.version)
    if err != nil {
        return true
    }
    cached, err := m.cache.ConfigHash()
    if err != nil {
        return true
    }
    if current != cached {
        return true
    }
    imageID, err := m.cache.ImageID()
    if err != nil || imageID == "" {
        return true
    }
    if _, _, err := m.client.ImageInspectWithRaw(ctx, imageID); err != nil {
        return true
    }
    return false
}
```

`fetchStatus` is a tea.Cmd that calls `mgr.Status(ctx)` and returns `statusUpdateMsg`.

- `statusUpdateMsg`: store info/err, schedule next tick: `tea.Tick(2*time.Second, func(t time.Time) tea.Msg { return tickMsg(t) })`

```go
RunE: func(cmd *cobra.Command, args []string) error {
    ctx, cancel := signal.NotifyContext(context.Background(), os.Interrupt, syscall.SIGTERM)
    defer cancel()

    cwd, err := os.Getwd()
    if err != nil {
        return fmt.Errorf("get working directory: %w", err)
    }

    cfg, _, err := config.LoadMerged(cwd)
    if err != nil {
        return fmt.Errorf("load config: %w", err)
    }

    c := cache.New(cwd)
    mgr, err := docker.NewManager(cfg, c, cwd, version)
    if err != nil {
        return err
    }

    jsonMode, _ := cmd.Flags().GetBool("json")
    plainFlag, _ := cmd.Root().PersistentFlags().GetBool("plain")

    // JSON and non-TTY paths: use existing plain text output
    if jsonMode || !tui.IsTTY(plainFlag) {
        info, err := mgr.Status(ctx)
        if err != nil {
            return err
        }

        if jsonMode {
            b, jErr := json.MarshalIndent(info, "", "  ")
            if jErr != nil {
                return fmt.Errorf("marshal JSON: %w", jErr)
            }
            fmt.Fprintln(cmd.OutOrStdout(), string(b))
            return nil
        }

        // Plain text output (existing code from lines 66-127)
        return printStatusPlain(cmd, cwd, cfg, info)
    }

    // TUI path -- use RunTUI for panic safety (D-27)

    if err != nil {

        return sv.Err

    case "stop":

    }

    return nil
},
```
